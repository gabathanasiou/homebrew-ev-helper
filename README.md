# homebrew-ev-helper

Homebrew tap for [ev-helper](https://github.com/gabathanasiou/ev-helper) — a blazing fast CLI for Pokémon FireRed/LeafGreen & R/S/E EV training.

**🌐 Web App:** [gabathanasiou.github.io/ev-helper](https://gabathanasiou.github.io/ev-helper/)

## Install

```bash
brew tap gabathanasiou/ev-helper
brew install ev-helper
```

## Usage

```bash
ev-helper                       # open the full TUI
ev-helper --pokemon=Pikachu     # jump straight to a Pokémon
ev-helper --route="Route 1"     # jump to a route
ev-helper --stat=attack         # browse by EV stat
```

## Keybindings

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate list |
| `→` / `Enter` | Select / drill in |
| `←` / `Esc` | Go back |
| `Tab` | Switch mode or cycle sort |
| `Ctrl+N` | Return home |
| `Esc` (home) | Quit |
